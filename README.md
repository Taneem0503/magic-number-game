# Magic Number Guessing Game

Interactive web-based game that guesses any number you think of between 1–100 using 7 “magic cards” and binary logic.

## Features
- Fully interactive with Yes/No buttons
- 7 cards generated automatically using binary numbers
- Clean, professional UI with restart option
- Presentation-ready for classroom or portfolio

## How to Play
1. Think of a number between 1 and 100
2. For each card, click **Yes** if your number appears, or **No** if it doesn’t
3. After 7 cards, the game reveals your number

## Technical Concept
- Each card represents a binary bit (1,2,4,8,16,32,64)
- Numbers appear on cards based on their binary representation
- Yes responses are added using bitwise logic to determine the final number
